## Project setup
`git clone https://github.com/gautemo/ordentlig-radio-app.git`

`npm install -g cordova`

### Platforms
To check current platforms
`cordova platform ls`

To add platforms
`cordova platform add ios/android/browser`

### Build app
`cordova build`
